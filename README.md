# 🎯 Real-Time Face Mesh Tracker

A real-time face mesh tracking system built using **OpenCV** and **MediaPipe** that detects and visualizes **468 facial landmarks** from a live webcam feed.

This project demonstrates practical implementation of **computer vision** for real-time facial analysis and can serve as a base for advanced applications like AR filters, emotion detection, and gesture-based interaction.

---

## 🚀 Features

* Real-time face detection using webcam
* 468 facial landmark tracking (Face Mesh)
* Smooth and efficient processing
* Lightweight and easy to run

---

## 🛠️ Tech Stack

* Python
* OpenCV
* MediaPipe

## ▶️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/face-mesh-tracker.git
cd face-mesh-tracker
```

### 2. Create virtual environment (recommended)

```bash
python3.10 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python app.py
```
